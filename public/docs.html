Quick Start

Get Shop with Friends running on your site in under 5 minutes.

Option 1: Script Tag (Easiest)

<!-- Add to your product pages -->
<script src="https://cdn.shopwithfriends.io/v1/swf.js"></script>
<script>
  ShopWithFriends.init({
    apiKey: 'your_api_key_here',
    productId: '12345'
  });
</script>

Option 2: NPM Package

npm install @shopwithfriends/sdk

import ShopWithFriends from '@shopwithfriends/sdk';

const swf = new ShopWithFriends({
  apiKey: 'your_api_key_here',
  apiUrl: 'wss://api.shopwithfriends.io'
});

await swf.init();

Core Features

Session Management

Create shopping sessions with unique invite links. Sessions automatically expire after 30 minutes of inactivity.

// Create a session
const session = await swf.createSession();
console.log(session.inviteLink);
// ‚Üí "https://yourstore.com/join/sess_abc123xyz"

// Join an existing session
await swf.joinSession('sess_abc123xyz');

Real-time Synchronization

All participants see the same products, cart updates, and scroll positions in real-time.

// Listen for sync events
swf.on('navigate', (data) => {
  console.log('Friend viewed:', data.product.name);
  // Navigate to the same product
  window.location.href = `/products/${data.product.id}`;
});

swf.on('cartUpdate', (data) => {
  console.log('Friend added to cart:', data.cart);
  // Update your cart UI
});

Reactions

Let shoppers react to products with fun emoji reactions.

// Send a reaction
swf.sendReaction('fire'); // üî•

// Listen for reactions
swf.on('reaction', (data) => {
  showReactionAnimation(data.reaction);
});

WebSocket API Live Now

Connect directly to our WebSocket server for custom integrations.

Connection

const ws = new WebSocket('wss://api.shopwithfriends.io');

ws.onopen = () => {
  // Connection established
  console.log('Connected to Shop with Friends');
};

Events

SEND CREATE_SESSION

Create a new shopping session and get invite link.

{
  "type": "CREATE_SESSION",
  "payload": {
    "metadata": {
      "storeName": "My Store"
    }
  }
}
SEND JOIN_SESSION

Join an existing session by ID.

{
  "type": "JOIN_SESSION",
  "payload": {
    "sessionId": "sess_abc123xyz",
    "userName": "John Doe"
  }
}
SEND SYNC_EVENT

Broadcast navigation, cart updates, or reactions to all participants.

{
  "type": "SYNC_EVENT",
  "payload": {
    "eventType": "NAVIGATE",
    "view": "DETAIL",
    "product": { "id": "123", "name": "Office Chair" }
  }
}

REST API

Use our REST API for server-side session management.

POST /api/sessions/create

Create a new session programmatically.

POST https://ap i.shopwithfriends.io/api/sessions/create
Content-Type: application/json

{
  "userId": "user_123",
  "userName": "Tola"

  "expiresAt": 1701236367890
}
GET /api/sessions/:sessionId

Get session details and participant list.
